# Kof — Offline First Table Ordering for Coffee Shops

Kof is a mobile ordering system for coffee shops. Customers scan a QR code on their table, browse the menu, and place orders directly from their phone — no app download required for the shop owner, no third-party platform fees.

---

## How it works

1. The coffee shop runs the **Kof Server** on a local machine or Raspberry Pi on their network.
2. Each table gets a printed QR code generated by the server.
3. Customers scan the QR code with the **Kof App**, browse the live menu, and place their order.
4. The order appears instantly on the shop's dashboard and the customer gets real-time status updates.

## Project Structure

```
Kof/
├── kof_app/      # Flutter mobile app (iOS + Android)

```

---

## kof_app — Mobile App

Built with Flutter. Supports iOS and Android.

**Features**
- QR code scanner to connect to a table
- Live menu with categories, availability and stock indicators
- Cart with order notes
- Real-time order status tracking via WebSocket
- Login / Register with email & password
- Google Sign-In
- Guest mode
- Forgot password flow
- Home screen with nearby coffee shop discovery (map)
- Push notification infrastructure (FCM)
- Multi-language: English, Português, Suomi
- Light / Dark / System theme
- Reveal-style side drawer

**Setup**

```bash
cd kof_app
flutter pub get
flutter gen-l10n
flutter run
```

Requires Flutter 3.x. See `kof_app/pubspec.yaml` for all dependencies.

For Google Sign-In and Maps to work you need:
- `android/app/google-services.json` — from your Firebase project
- `ios/Runner/GoogleService-Info.plist` — from your Firebase project
- A Google Maps API key in `AndroidManifest.xml` and `AppDelegate.swift`

---

## kof_server — Backend Server

Built with Node.js + Express + SQLite. Designed to run on a Raspberry Pi or any local machine on the coffee shop's network.

**Features**
- Menu management (items, categories, availability, stock)
- Table session management with QR code generation
- Order creation and real-time status updates via WebSocket
- Admin dashboard (web UI)
- Automatic database backups
- Rate limiting and basic auth protection

**Setup**

```bash
cd kof_server
npm install
cp .env.example .env   # then fill in your values
node src/index.js
```

See `kof_server/DEPLOYMENT.md` for full deployment instructions including Raspberry Pi setup.
